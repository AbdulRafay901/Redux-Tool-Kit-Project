# 📖 About The Project

**Media Search** is a modern media discovery application built using **React.js and Redux Toolkit**.

The application integrates with external APIs to fetch high-quality photos and videos, providing users with a smooth search experience through dedicated media sections.

The project focuses on efficient state management, reusable components, API handling, and creating a clean user experience similar to real-world media platforms.

# ✨ Features

## 🔍 Smart Media Search

- Search photos and videos instantly
- Dynamic API-based results
- Fast and responsive search experience
- Real-time result updates

# 🚀 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/AbdulRafay901/Redux-Tool-Kit-Project.git
```

Navigate to project directory:

```bash
cd media-search
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Create production build:

```bash
npm run build
```
